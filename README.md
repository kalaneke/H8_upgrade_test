# 软件升级工具项目策划书

## 一、项目概述

### 1.1 项目背景

为满足设备软件远程维护与版本迭代需求，开发一套完整的软件升级工具，实现IAP固件升级与资源文件同步更新功能。

### 1.2 项目目标

- 实现IAP固件文件的自动检测与升级
- 实现资源文件的增量更新与同步
- 提供稳定、可靠、安全的升级流程

---

## 二、功能模块设计

### 2.1 IAP文件升级模块

#### 2.1.1 版本检测

| 序号 | 功能项         | 说明                                               |
| ---- | -------------- | -------------------------------------------------- |
| 1    | 读取本地版本   | 读取磁盘IAP目录下的VER.DAT文件，获取当前系统版本号 |
| 2    | 获取服务器版本 | 连接版本服务器，查询最新版本信息                   |
| 3    | 版本比对分析   | 对比本地与服务器版本，确定需要升级的部件列表       |

#### 2.1.2 升级流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 读取VER.DAT │ → │ 版本比对    │ → │ 确定升级项  │
└─────────────┘    └─────────────┘    └──────┬──────┘
                                             ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 升级完成    │ ← │ 下载新文件  │ ← │ 清理IAP目录 │
└─────────────┘    └─────────────┘    └─────────────┘
```

#### 2.1.3 详细步骤

1. **读取版本信息**
   
   - 路径：`IAP/VER.DAT`
   - 格式：文本/二进制版本标识
2. **版本比对**
   
   - 连接版本服务器API
   - 获取最新版本清单
   - 逐项比对各部件版本
3. **清理目录**
   
   - 保留 `VER.DAT` 文件
   - 删除IAP目录下其他所有文件
4. **下载更新**
   
   - 根据比对结果，下载需要升级的部件文件
   - 保存至IAP目录
   - 校验文件完整性

---

### 2.2 资源文件更新模块

#### 2.2.1 资源类型

| 资源类型 | 文件格式   | 存放路径    |
| -------- | ---------- | ----------- |
| 声音文件 | .mp3/.wav  | RES/audio/  |
| 说明书   | .pdf/.docx | RES/manual/ |
| 配置文件 | .json/.xml | RES/config/ |

#### 2.2.2 更新机制

1. **文件比对**
   
   - 计算本地文件哈希值（MD5/SHA256）
   - 与服务器文件哈希进行比对
   - 生成差异文件清单
2. **增量下载**
   
   - 仅下载哈希不一致的文件
   - 支持断点续传
   - 下载完成后校验完整性
3. **本地更新**
   
   - 备份原有文件（可选）
   - 替换为新版本文件
   - 更新本地文件索引

## 四、接口设计

### 4.1 服务器接口

#### 获取版本信息

```
GET /api/version/latest
Request:
{
    "deviceType": "string",
    "currentVersion": "string"
}

Response:
{
    "version": "v2.1.0",
    "components": [
        {
            "name": "main_firmware",
            "version": "v2.1.0",
            "url": "https://server/iap/main_v210.bin",
            "checksum": "sha256:xxxx"
        }
    ]
}
```

#### 获取资源清单

```
GET /api/resources/manifest
Response:
{
    "resources": [
        {
            "path": "audio/welcome.mp3",
            "checksum": "md5:xxxx",
            "size": 102400,
            "url": "https://server/res/audio/welcome.mp3"
        }
    ]
}
```

## 六、风险与对策

| 风险         | 影响     | 应对措施                 |
| ------------ | -------- | ------------------------ |
| 升级中断     | 系统异常 | 支持断点续传，升级前备份 |
| 网络异常     | 下载失败 | 增加重试机制，超时处理   |
| 版本不兼容   | 无法启动 | 版本校验，回滚机制       |
| 存储空间不足 | 升级失败 | 升级前检查磁盘空间       |
